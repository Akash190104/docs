# Supported Markdown Features

This table shows the Azure DevOps Wiki markdown features that are supported by Work Item Wiki.

| Feature             | Supported         |
| ------------------- | ----------------- |
| Header (H1, H2, H3) | ✅                |
| Italic              | ✅                |
| Bold                | ✅                |
| Link                | ✅                |
| Attachment          | ✅                |
| Image               | ✅                |
| Code / Code Block   | ✅                |
| Unordered list      | ✅                |
| Ordered list        | ✅                |
| Table               | ✅                |
| Mermaid Diagram     | ❌                |
| Work Item Mentions  | ❌                |
| Table of Contents   | ❌                |
| Formulas            | ❌                |
| Mention             | ❌                |
| Query Results       | ❌                |
| Task List           | ❌ (Display only) |
